# 🎓 Lost in Transition: A Global Study on Student Dropout Rates

This repository features an end-to-end data analytics project exploring the "dropout cliff"—the significant decline in student enrollment during transitions from primary to secondary and tertiary education. 

Using **World Bank Education Statistics**, this project identifies global retention gaps with a specialized focus on **India** as a recurring case study.

## 🚀 Project Overview
The objective is to quantify student attrition across different income groups and regions, identifying where the educational system loses the most students and analyzing the survival index of students globally.

## ✨ Key Features
* **Global Comparative Analysis:** Comparing enrollment trends across 200+ countries.
* **The "Dropout Cliff" Identification:** Visualizing the steep decline in transitions using bivariate scatter plots.
* **Survival Index Modeling:** A custom metric engineered to rank countries based on their ability to retain students through tertiary education.
* **India Case Study:** Detailed deep-dive into India's educational progress compared to regional peers.
* **Interactive Dashboard:** A Power BI dashboard providing a granular view of dropout risks.

## 🛠 Technologies Used
* **Python (Pandas, NumPy):** Data cleaning and feature engineering.
* **Matplotlib & Seaborn:** Statistical data visualization.
* **Power BI:** Interactive dashboarding and storytelling.
* **Jupyter Notebook:** For documented Exploratory Data Analysis (EDA).

## 🧠 Project Workflow
1. **Data Acquisition:** Sourcing multi-year enrollment data from the World Bank.
2. **Preprocessing:** Handling missing values and merging primary, secondary, and tertiary datasets into a unified `cleaned_education_dataset.csv`.
3. **Feature Engineering:** Creating the *Education Survival Index* and *Dropout Risk Levels*.
4. **EDA:** 6-Milestone analysis covering Univariate, Bivariate, and Multivariate trends.
5. **Reporting:** Final synthesis of findings in a comprehensive academic report.

## 📊 Dashboard Highlights
* **Dropout Risk Level:** Categorization of countries into High, Moderate, and Low risk.
* **Retention Trends:** Year-over-year enrollment growth analysis.
* **Regional Benchmarking:** Comparing India’s performance with South Asia and global averages.

## 📁 File Descriptions
* `notebooks/Lost_in_Transition.ipynb`: The full Python analysis and visualization code.
* `data/`: Contains the raw and cleaned datasets.
* `reports/DataDragons_Dashboard.pbix`: The interactive Power BI file.
* `reports/Data_Dragons_Report.docx`: The full academic project report.

## 📌 Conclusion
This project demonstrates proficiency in the full data science lifecycle—from raw data extraction to business intelligence reporting. It highlights the critical need for targeted interventions at the secondary education level to prevent the "dropout cliff."
